# Easy-POST
**POST送信をお手軽に**  
というのがコンセプト(?)  
Sendボタンをクリックすると、テキストボックスの内容を元にJSONフォーマットのテキストを作成し、指定されたURLにPOST送信します。  
使用するためには、JavaScriptを有効にしてください。
